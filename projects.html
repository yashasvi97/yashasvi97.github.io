Featured Projects:

  • Smart Glasses - A prototype to help the visually impaired understand the text present in front of them. This project was featured in Delhi Mini Maker Faire. Project link, code
  • Periocular Recognition - A classifier to recognize people from their periocular region using a combination of features like Local Binary Patterns(LBP), Histogram of Oriented Gradients(HOG) and Scale Invariant Feature transform(SIFT). Code available at github
  • BiTriplet Loss - Currently working on an extension of the famous Triplet Loss to incorporate heterogenous face recognition using metric learning. Working with images from different domains like VIS-NIR or high-low resolution image pairs.
  • Landslide Prediction - Made a landslide prediction model for Mandi-Manali Route using last 10 year rainfall data. Looked at various preprocessing techniques to remove class imbalance in the dataset to build a better model. I did this work while working as a research intern at IIT Mandi. Other part of the internship was aimed at building a weather station which logs the neighbourhood temperature, humidity, soil moisture values to an online server using Arduino.
  • Multi Heuristic A* - AI course project invloving a comparison of the classical search algorithm A* and its variant Multi Heuristic A* (MHA*). Comparison for multiple uses cases was shown. Code available at github
  • Trust based recommender system - Worked on building a recommender system which incorporates the social interactions between users in the system. A trust graph was made using edge weights as either +1 or 0. Our model performed better than the normal user-user recommender system. Code available at github
  • DBLP search engine - Built a Java based search engine to analyse, search and query the entire DBLP data in XML format. The engine could also predict the number of publications of a particular author for next 5 years.
  • File Sync - A command line utility to share and upload files between multiple users. The project involved working with Lua scripts to help syncing the shared data. Code available at github
  • NBA record query system - As a part of DBMS course project, built a Java applet which searches the NBA record dataset and solves multiple SQL queries on it.
  • Pentesting on intra-iiitd servers - Pentested various intranet servers like the it-helpdesk, library server and the admissions portal to check for any vulnerabilities.
